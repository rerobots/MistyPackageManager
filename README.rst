Misty Package Manager (mpm)
===========================

Getting Started
---------------

Create an empty skill named "demo"::

  mpm init demo

The resulting JavaScript (JS) file and meta file are in the directory src/. When
ready, create a bundle::

  mpm build

and upload ``dist/demo.zip`` to a Misty robot.
